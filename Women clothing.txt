1. Изменить валюту на сайте
-В шапке страницы нажать на выбор страны
-Выбрать вид валюты 
-Обновить настройки

Результат : Все ценники сайта будут показывать выбранную валюту --<span class="u-weight--bold u-padding-l--md">$USD</span>

2. Добавление товара в корзину
-В поиск ввести название товара
-В результате поиске выбрать свой товар
-Выбрать размер
-Добавить в корзину

Результат : Выбранный товар находится в корзине

3. Изменение количества товаров в корзине
-Перейти в корзину
-В вещи в корзине выбрать "изменить"
-Изменить количество
-Обновить

Результат : Выбранный параметр вещи изменён --<span class="sb_display">2</span>

4. Удаление товара из корзины
-Перейти в корзину
-Найти необходимый товар
-Нажать убрать

Результат : Выбранный товар удалён из корзины

5.Проверка бесплатной доставки
-Главная
-Добавляем товары на сумму от 100 usd
-Переходим в корзину

Результат : В обзоре заказа "Бесплатная доставка!" 

6.Оформление заказа с пустой корзиной
-Главная
-Корзина
-Удалить все товары
-Оформить заказ

Результат : После удаления товаров кнопка оформление исчезла

 7.Оформление заказа с товарами в корзине без введения данных
-Главная
-Выбрать и добавить товар
-Оформить заказ как гость
-Сохранить и продолжить

Результат : При попытке оформить заказ обязательные поля изменили цвет --required aria-invalid="true"

8.Оформление заказа с некоректными личными данными
-Главная
-Выбрать и добавить товар
-Оформить заказ как гость

-Перейти к оплате
-Ввести неправильный номер карты

Результат : Поле номера карты изменило цвет, в теге input изменились атрибуты data-validate-not-empty ("true"), aira-invalid ("true").

-Корзина

-Далее
-Ввести данные

Результат : При вводе некорректных данных поля подствечиваются и появляется строка "Обязательное поле"
